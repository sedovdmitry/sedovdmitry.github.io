Два Подростка

Адаптивный спецпроект для центра Молодежная инициатива, 2013

Дмитрий Седов 2013-2015

Используемые технологии: Bootstrap, HTML5, CSS3, PHP, Picturefill.

Спецпроект для центра "Молодежная инициатива" о половом воспитании подростков.

Оптимизирован под поисковые системы и мобильные устройства, быстрая загрузка для моибльных устройств за счет отправки разных по размеру картинок в зависимости от экрана пользователя. Например, большие картинки отправляются устройствам с экраном выше 600 пикселей, средние -- выше 375, маленькие -- меньше 375 пикселей (Picturefill).

Главная картинка ведет себя точно также, но реализовано это с помощью CSS3.

Адаптивный лендинг фитнес-браслетов, 2015

Дмитрий Седов 2015

Используемые технологии: Bootstrap, JavaScript, HTML5, CSS3, PHP.

Адаптивная посадочная страница для заказа фитнес-браслетов MiBand.

Главная (самая большая и тяжелая) картинка заменяется на картинку меньшего размера в засисимости от размера экрана пользователя (CSS3).

Плавная прокрутка по странице между доставкой, оплатой и заказом товара.
